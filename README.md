# vue-antd-pro
  
use [Ant Design of Vue 1.1.0](https://vuecomponent.github.io/ant-design-vue/docs/vue/introduce-cn/)  
  
## Project setup  
```  
yarn install  
```  
  
### Compiles and hot-reloads for development  
```  
yarn run serve  
```  
  
### Compiles and minifies for production  
```  
yarn run build  
```  
  
### Lints and fixes files  
```  
yarn run lint  
```
